**Option B: Run Locally**
```bash
cd trip-adventurers
npm install
npm run dev
```
Open `http://localhost:5173` in your browser.

## Walkthrough

** The following will help with initial start ups. If you would like a more in-depth viewing, please follow the link of our video demo: https://youtu.be/OxL9jPsUaNQ
1. **Login** - Enter the credentials above and click "Sign In"

2. **View Trip List** - Two sample trips are available:
   - **"Holidays in YYC"** - Fully implemented (use this one!)
   - **"Raptors in Toronto"** - Demonstrates view-only member permissions

3. **Explore Trip Features** - Click on "Holidays in YYC" to:
   - View Calgary-based events
   - Sort/Filter events in the Explore tab
   - Check your day's itinerary
   - Create custom events
   - View upcoming weather and news

4. **Create Your Own Trip** (Optional) - Click "the top right plus button" in the itinerary page to test trip creation
