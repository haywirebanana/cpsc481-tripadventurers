# Trip Adventurers

A trip management application built with React and Vite (Deployable through Vercel)

## Requirements

- Node.js
- npm

## Setup & Run
```bash
# Clone the repository
git clone https://github.com/haywirebanana/cpsc481-tripadventurers.git

# Navigate to project (if not already)
cd cpsc481-tripadventurers/trip-adventurers

# Install dependencies
npm install
```

# Deployment Steps
## Option A: Locally (Mainly for testing and seeing real-time changes)
```
npm run dev
```
Open `http://localhost:5173` in your browser.

### Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Option B: Vercel Website (Will be mainly used for final deliverable which needs finalized commits/features)
https://trip-adventurers.vercel.app/
